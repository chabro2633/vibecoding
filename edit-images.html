📝 편집 방법

1. 이미지 교체: 각 이미지의 src 속성을 원하는 이미지 경로로 변경하세요.

2. 크기 조정: style="max-width: 600px;" 부분을 수정하여 크기를 조절하세요.

3. 설명 변경: alt 속성과 설명 텍스트를 수정하세요.

4. 저장 후 적용: 편집 완료 후 변경사항을 Next.js 파일에 반영하세요.
